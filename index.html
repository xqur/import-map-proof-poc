ℹ️ What This Demonstrates

The file proof.js is an external JavaScript file that, when loaded in a browser, demonstrates that an import map override succeeded — i.e., that a page loaded a script from a controlled origin instead of the intended origin.

Expected Behavior

  • ✅ A console message appears: "Import map override proof: external JavaScript executed successfully"
  • ✅ A browser alert dialog appears: "PoC successful: external JavaScript module executed"
  • ❌ No data is collected or transmitted
  • ❌ No cookies are accessed
  • ❌ No network requests are made by proof.js

Usage

To verify this proof file works as a standalone resource, you can load it via the browser console:

// Create a script element
var s = document.createElement('script');
s.src = 'https://xqur.github.io/import-map-proof-poc/proof.js';
document.head.appendChild(s);

Or use import in supported environments:

import('https://xqur.github.io/import-map-proof-poc/proof.js');

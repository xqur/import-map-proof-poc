When a user visits this page, it makes cross-origin requests to OKX APIs. The server reflects access-control-allow-origin: *, allowing any website to read the response.

💀 Exploit Chain

Vulnerability: CORS misconfiguration allowing any origin

Impact:

  • Any malicious website can make authenticated API calls to OKX
  • If a victim is logged in with API key cookies/headers, attacker can read account balance, orders, config, positions
  • CSRF-style attacks possible against OKX users

Affected endpoints confirmed:

  • /api/v5/account/balance
  • /api/v5/account/config
  • /api/v5/trade/order
  • /api/v5/trade/orders-pending
  • /api/v5/account/positions
